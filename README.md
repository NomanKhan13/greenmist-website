### 🏨 GreenMist: Luxury Hotel Booking Platform (Customer-Facing)

**Overview**
GreenMist is a premium, full-stack hotel booking platform designed to deliver a seamless and visually cinematic user experience. Built with a focus on high-fidelity aesthetics and performance, the application allows users to explore luxury properties, manage secure bookings, and experience a smooth, intuitive checkout process. The UI is custom-crafted to reflect a nature-inspired brand identity, utilizing minimalist design principles to elevate the customer journey from discovery to reservation.

**Key Features**
* **Cinematic Property Discovery:** Dynamic, high-resolution property listings with advanced filtering capabilities, optimized for fast loading and SEO.
* **Seamless Booking Flow:** A custom-built, interactive booking drawer featuring real-time date-range validation and availability checking.
* **Secure Authentication:** Protected user routes and seamless onboarding flow for guest profile management and reservation tracking.
* **Responsive Design System:** A fully custom, mobile-first interface emphasizing clean typography, deep forest greens, and mist-grey color palettes.

**Technical Architecture**
* **Framework:** Next.js (App Router) for Server-Side Rendering (SSR) and optimal search engine visibility.
* **Database & Backend:** Supabase (PostgreSQL) with strictly enforced Row Level Security (RLS) to protect user and booking data.
* **Styling:** Tailwind CSS integrated with Shadcn UI for accessible, high-quality component architecture.
* **State Management:** Custom React hooks for handling complex, multi-step booking logic.
